# Requirement
- gnu stow
- hyprland
- waybar
- git
- hyprpaper

## Installation
```bash
pacman -S stow
```
```bash
pacman -S git
```
```bash
pacman -S hyprpaper
```
```bash
pacman -S waybar
```

## Description
The Hyprland dir not only contains the hyprland config but also the theming for hyprpaper.

## Theme

- maincolor #ACF079
- secondary #0F2523
- third #6ca65a
